# 勝手にデジタル庁 Project

「紙と対面訪問、もう要らないかも？」を調べるプロジェクト

---

## 📋 このプロジェクトについて

**勝手にデジタル庁**は、公共サービス（国・都道府県・市区町村などの手続き）が、**実際にどこまでオンラインで完結できるのか**を調べて、みんなでシェアする知識共有プロジェクトです。

「窓口に行かなきゃダメ」と思っていた手続きが、実はオンラインでできたり、逆に「ネットでできる」と聞いたのに結局窓口に行く羽目になったり……。そんな**情報のバラバラ感**を解消したいという想いから生まれました。

---

## 🤔 なぜこのプロジェクトを始めたのか

### 公共サービスのDX状況が、とにかくわかりにくい

政府、東京都、各都道府県、市区町村……公共機関の手続きが「どこまでデジタル化（DX）されているのか」を調べるのは、正直とても大変です。

- 詳しくホームページを探し回る
- 窓口に問い合わせる
- 電話で長時間待たされる

こうした手間をかけないと、**本当のところがわからない**のが現状です。

### 実際に体験した"すれ違い"

例えば、法人税や消費税の納税証明を取得する必要があったとき、調べた結果「窓口に行く必要がある」と出てきました。

そこで実際に税務署に行ったところ、「e-Taxがあるのに、なんでそっちを使わないんですか？」と言われてしまいました。

税務署の方が悪いわけでは全くありません。せっかく便利な仕組みがあるのだから、それを普及させたいという正しいミッションのもとで対応されていたのだと思います。

**みんな真面目に仕事をしているのに、結果として無駄が生まれてしまう。**

この「もったいない状況」をなんとかしたい、と思いました。

### 縦割り構造が生む情報の断絶

公共機関は、どうしても縦割り構造になりがちです。国税庁、都税事務所、区役所……それぞれが独立して動いているため、情報が一箇所にまとまっていません。

根本的な解決は難しいかもしれませんが、**「もう待っていられない」**という気持ちから、このプロジェクトを立ち上げました。

### 自分のために、そしてみんなのために

まずは私自身のために作ってみて、便利であればどんどん広げていけたらいいな、という軽い気持ちで始めた、小さな試みです。

---

## 🔍 現在調査中のテーマ

このプロジェクトでは、実際の手続きを一つひとつ検証し、「オンラインで完結できるか（DX済み）」「窓口が必要か（NO-DX）」を記録しています。

### 📌 調査済み・調査中のトピック

| トピック | DX状況 | 説明 |
|---------|--------|------|
| **法人の法人税・消費税の納付証明書取得** | ✅ YES-DX | オンラインで取得可能 |
| **法人の都税納税証明書取得** | ❌ NO-DX | 窓口訪問が必要 |
| **自社及び他社の簿本（登記簿謄本）取り寄せ** | ✅ YES-DX | オンラインで取得可能 |

※ 随時更新予定

## 💡 こんな人に役立ちます

- **個人事業主・フリーランスの方**:  確定申告や各種証明書の取得を効率化したい
- **中小企業の経営者・総務担当者**: 会社の手続きをできるだけオンラインで済ませたい
- **行政手続きを頻繁に行う方**:  無駄な窓口訪問を減らしたい
- **デジタル化に関心がある方**: 公共サービスのDX状況を知りたい
